# The Enterprise Glossary

**A Compendium of SAP, ERP, and Product Costing Terms**

## Core SAP & Costing

- **ACRU (Actual Costing):** SAP application for calculating actual costs of materials and production.
- **BOM (Bill of Materials):** The "recipe" for manufacturing, containing all components and routings.
- **COPA (Controlling-Profitability):** Profitability analysis module used for detailed financial reporting.
- **FERT:** SAP material type for Finished Goods.
- **ROH:** SAP material type for Raw Materials.
- **PUP (Periodic Unit Price):** The calculated price for a material within a specific period in SAP Actual Costing.

## Data & Infrastructure

- **D@T (Data @ Tyson):** Collaborative initiative for migrating BI from EAP to Google Cloud Platform (GCP).
- **EAP (Enterprise Analytics Platform):** The core landscape for Tyson's data warehousing.
- **GCP (Google Cloud Platform):** Used for BigQuery and advanced data management.
- **OLAP vs. OLTP:** Analyzing database info for multi-systems (Analysis) vs. Administering day-to-day transactions (Processing).

## Product Management & Delivery

- **CAB (Change Advisory Board):** The governing body that approves system changes and transports.
- **INVEST:** Criteria for high-quality user stories (Independent, Negotiable, Valuable, Estimable, Small, Testable).
- **SIT (System Integration Testing):** Verifying the behavior of the complete integrated system.
- **UAT (User Acceptance Testing):** Final validation by end-users before production release.
